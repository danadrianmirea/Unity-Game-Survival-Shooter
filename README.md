## Deskripsi Aplikasi
Extended Survival Shooter merupakan modifikasi dari game Survival Shooter yang memiliki tambahan fungsionalitas sebagai berikut:
1. Story Mode
2. Save & Load Game
3. Game Over
4. Local Scoreboard
5. Main Menu
6. Shopkeeper
7. Weapon
8. Pet
9. Cheat
10. Weapon Upgrade (**BONUS**)

## Library yang Digunakan
- Unity
- UnityEngine
- System
- TMPro
